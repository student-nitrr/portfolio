    NLP & Machine Learning Project

    Duplicate Question Predictor

    Tech Stack: Python, Machine Learning, NLP, Streamlit, Scikit-learn, Pandas, NumPy, Feature Engineering, Pickle Deployment

    Developed an NLP-based Duplicate Question Prediction system that determines whether two user-entered questions have the same semantic meaning despite different wording.

    • Implemented advanced text preprocessing techniques including tokenization, stop-word removal, stemming, and text normalization to improve prediction accuracy.
    • Engineered multiple linguistic and statistical features such as word overlap, common words ratio, token-based similarity, and semantic matching scores for effective duplicate detection.
    • Trained and evaluated machine learning models on question-pair datasets to classify whether two questions are duplicates or non-duplicates.
    • Built an interactive web application using Streamlit that allows users to input question pairs and receive real-time duplicate prediction results.
    • Designed an end-to-end ML pipeline covering data preprocessing, feature extraction, model training, prediction, and deployment.
    • Deployed the application on Streamlit Cloud, enabling public accessibility and real-time inference through a user-friendly interface.
    • Applied Natural Language Processing techniques to solve a real-world information retrieval problem commonly faced by platforms such as Quora and Stack Overflow.
