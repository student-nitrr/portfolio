1. Open the Project in VS Code

Open the folder containing your project.

2. Open Terminal

Press Ctrl + ` (backtick) or go to Terminal → New Terminal.

3. Initialize Git
git init 

4. Add All Files
git add .

5. Commit the Files
git commit -m "Initial commit"

6. Create a New Repository on GitHub
Go to GitHub
Click New Repository
Enter repository name
Click Create Repository
Do not add README if your local project already has files

7. Connect Local Project to GitHub

Copy the repository URL and run:

git remote add origin https://github.com/YOUR_USERNAME/REPO_NAME.git

Example:

git remote add origin https://github.com/sakshamsingh7376/my-project.git

8. Push the Code

For a new repository:

git branch -M main
git push -u origin main

9. Verify

Refresh your GitHub repository page. Your project files should now be visible
